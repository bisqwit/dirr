1. Purpose

List directories and files in *nix system. A colourful alternative for /bin/ls . Not completely compatible with it, though.
Given some luck, may compile on DOS too.

5. Downloading

The official home page of dirr is at http://iki.fi/bisqwit/source/dirr.html.
Check there for new versions.

Additionally, the most recent source code (bleeding edge) for dirr can also be downloaded by cloning the Git repository by:

  • git clone git://bisqwit.iki.fi/dirr.git
  • git checkout origin/release -b release
  • git checkout origin/master -b master
